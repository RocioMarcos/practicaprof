## Dashboard de Análisis de Tráfico Web - DGIPSE
#### Descripción
Este proyecto es una aplicación web desarrollada con Streamlit para el análisis inteligente del tráfico web del portal institucional del gobierno provincial de Santiago del Estero. La herramienta permite identificar patrones de acceso, detectar anomalías y optimizar el rendimiento del sitio dgipse.gob.ar.

#### Objetivos
Automatizar el análisis del tráfico web

Detectar patrones anómalos y posibles bots

Visualizar métricas clave en tiempo real

Segmentar usuarios por comportamiento

Generar insights accionables para optimización

#### Características Principales
#### Métricas en Tiempo Real
Usuarios únicos y total de requests

Distribución por dispositivos y navegadores

Análisis geográfico del tráfico

Porcentaje de anomalías detectadas

#### Visualizaciones Interactivas
Tráfico por hora del día

Distribución geográfica (mapa de calor)

Top 10 páginas más visitadas

Segmentación por dispositivo y navegador

#### Detección Inteligente
Algoritmo Isolation Forest para detección de anomalías

Clustering K-Means para segmentación de usuarios

Identificación automática de patrones sospechosos

Detección de bots y scraping automatizado

#### Tecnologías Utilizadas
Frontend: Streamlit

Análisis de datos: Pandas, NumPy

Visualización: Matplotlib, Seaborn

Machine Learning: Scikit-learn

Procesamiento: Python 3.8+
